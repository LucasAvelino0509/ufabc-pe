## Soma 1 + 2 + ... + n

Faça uma função recursiva para calcular a soma dos primeiros n inteiros positivos 1 + 2 + ... + n. Sua função deve ter o cabeçalho:
`int soma(int n);`

 
Esqueleto de programa:

    #include <stdio.h>
    
    int soma(int n) {
    	// seu código aqui ...
    }
    	int main() {
    	int n;
    	scanf("%d", &n);
    	printf("%d\n", soma(n));
    	return 0;
    }
